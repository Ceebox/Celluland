About

Celluland is a little JavaScript toy to meddle around with different Cellular Automata algorithms.

Have a look through some of the presets if you want to get started!

API

Cell

Contains the state of the current cell and also functions for interacting with other cells.

cell.state

The colour of the current cell.

cell.x

The X (horizontal) position of the current cell.

cell.y

The Y (vertical) position of the current cell.

cell.phase

The current iteration count. This can be useful for creating things programmatically.

Colour

Contains all the possible colours a cell can be.

NOTE: A cell can return either a boolean (black / white), a colour, or a number corresponding to a colour value.

colour.

  • BLACK: 0
  • WHITE: 1
  • GRAY: 2
  • RED: 3
  • GREEN: 4
  • BLUE: 5
  • YELLOW: 6
  • ORANGE: 7
  • PURPLE: 8
  • CYAN: 9
  • PINK: 10
  • BROWN: 11
  • DARK_GREEN: 12
  • NAVY_BLUE: 13

Different colours that can be returned to colour a cell.

Celluland

Docs

About

Celluland is a little JavaScript toy to meddle around with different Cellular Automata algorithms.

Have a look through some of the presets if you want to get started!

API

Cell

Contains the state of the current cell and also functions for interacting with other cells.

cell.state

The colour of the current cell.

cell.x

The X (horizontal) position of the current cell.

cell.y

The Y (vertical) position of the current cell.
